in order to run this example, you need to set the following environment variables:

- BASE_URL
- STARKNET_RPC
- PRIVATE_KEY
- ACCOUNT_ADDRESS

copy the .env.template file to .env and fill in the values.
```bash
cp .env.template .env
```

install the dependencies
```bash
pnpm install
```

run the example
```bash
npx ts-node src/index.ts
```
